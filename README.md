# MainQuest02 - Telecom Churn Analysis

## 1. 문제 정의 및 데이터 이해
통신사 고객 이탈을 예측하여 선제적 유지 전략 수립을 목표로 한다.

## 2. 데이터 전처리
결측치 및 이상치를 확인하고, 범주형 변수는 인코딩 처리하였다.

## 3. EDA
고객센터 문의 횟수와 International plan이 이탈과 높은 연관을 보였다.

## 4. 모델 학습
Logistic Regression, Random Forest, KNN 모델을 학습하였다.

## 5. 모델 평가
Recall과 F1-score를 중심으로 모델 성능을 비교·평가하였다.

## 6. 모델 해석
Customer service calls가 가장 중요한 이탈 예측 변수로 나타났다.

## 7. 비즈니스 인사이트 및 액션 플랜
고위험 고객 대상 맞춤 유지 전략을 제안하였다.

## 8. 문서화 및 발표 준비
분석 결과를 문서화하고 발표 자료로 정리하였다.

## Dataset
- Telecom Churn Dataset (Kaggle)
- https://www.kaggle.com/datasets/mnassrib/telecom-churn-datasets
